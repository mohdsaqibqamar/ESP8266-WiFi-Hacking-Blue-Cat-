Installation Guide

Follow these simple steps to install the firmware via Arduino IDE.

1

Get the Software

Download and install the official Arduino IDE on your computer.

2

Add the Spacehuhn Core

In Arduino, go to File > Preferences. Find the Additional Boards Manager URLs box and paste this link:

https://raw.githubusercontent.com/SpacehuhnTech/arduino/main/package_spacehuhn_index.json
3

Install the Board

Go to Tools > Board > Boards Manager. Search for "deauther" and install the package named ESP8266 Deauther.

4

Select Board & LwIP

Go to Tools > Board and select Generic ESP8266 Module. Then go to Tools > LwIP Variant and select v2 Higher Bandwidth (Crucial for the repeater to work fast).

5

Upload the Code

Plug in your ESP8266 via USB. Select its Port in Tools > Port. Finally, click the Upload button (the right arrow) and wait for it to finish!
